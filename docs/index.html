一行から始められる

依存パッケージなし。Python 標準ライブラリだけで動きます。

Python

from logballoon import LogBalloon

lb = LogBalloon(
    app_name="FFT Analyzer",
    version="1.0.0",
    endpoint="https://example.com",
)
lb.start()
lb.event("export_complete", {"rows": 120})
  • installation_id の自動生成
  • 起動ログ / 任意イベント / クラッシュ捕捉
  • Offline queue（SQLite）
  • HTTP 送信は urllib（追加依存なし）
  • Windows 10/11・Linux・ラズパイ想定
